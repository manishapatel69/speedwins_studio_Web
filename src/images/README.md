# Images Required for Speedwins App

Add the following image files to the `/src/images` folder:

## Required Images

1. **speedwins-logo.png**
   - The Speedwins logo for the navbar
   - Recommended size: 200px width (will be scaled to 40px height)
   - Format: PNG with transparency preferred

2. **google-play-icon.png**
   - Google Play Store badge/icon
   - Recommended size: 64px × 64px
   - Format: PNG with transparency

3. **app-store-icon.png**
   - Apple App Store badge/icon
   - Recommended size: 64px × 64px
   - Format: PNG with transparency

## How to Add Images

1. Export/prepare your images in the formats above
2. Place them in: `src/images/` folder (already created)
3. The app will automatically reference them in the JSX code

## Image Paths in Code

- Logo: `/src/images/speedwins-logo.png`
- Google Play: `/src/images/google-play-icon.png`
- App Store: `/src/images/app-store-icon.png`
